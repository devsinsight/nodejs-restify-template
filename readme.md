# Installation

Create a folder for your project:

```sh
$ mkdir MyProjectName
$ cd MyProjectName
```
Clone the repository:
```sh
$ git clone https://github.com/orgcompany/nodejs-restify-template
```

# Development

grunt watch:
```sh
$ npm run watch
```

# Build

Development environment:
```sh
$ npm run build:dev
```

Production environment:
```sh
$ npm run build:prod
```
